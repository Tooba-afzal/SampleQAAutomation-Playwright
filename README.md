# SampleQAAutomation-Playwright
This is a sample repository to automate the https://demoqa.com/ website
- After clonning the repo, execute command
  npm i

 // Move to Desktop
   cd Desktop/

 //Check Status
   git status

 // To Add something
    git add "-----"

 // To Add all changes
   git add .

 // Commit changes
  git commit -m "...."

// To Push changes in Repo
  git push url

 // Move One Step Back
   cd .

 // Back to Homepage
   cd ..

 // Create New Branch
   git branch "branch name"

 // Checkout branch
   git checkout "branch name"
